# Peach Paper

A rich-text note-taking application built with Flutter, inspired by a YouTube tutorial and enhanced with several personal improvements and refinements. This is my final project for my flutter bootcamp

---

## Overview

This Notes App enables users to create, edit, and manage notes with a clean, modern interface. It includes authentication, a rich-text editor, and consistent design built with Material 3.

While the project is based on a YouTube tutorial (credit below), I used it as an opportunity to:

- Deepen my understanding of Flutter architecture  
- Improve UI consistency  
- Add my own design and development workflow enhancements  
- Apply best practices for clean, maintainable code

---

## Features

- User Authentication (Firebase)  
- Rich Text Editing (Quill)  
- Clean, consistent UI built with Material 3  
- State Management using Provider  
- Widget-based architecture for reusability  
- Git Hooks for automated pre-commit checks  
- Custom fonts (Poppins and Fredoka)
